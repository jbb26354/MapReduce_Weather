Project # 1: Data Processing with MapReduce

Due Thursday, February 12th 2015, before class starts.

This is a group project (two students per group). Weight: 15% of class grade.

PART I (60%)

In this assignment you will use the Virtual Machine (Linux + Hadoop) used in class to implement two MapReduce programs. Both programs will use the Meteorological Station Data Set presented in class and also available in the VM.

Create MapReduce programs to solve the following problems:

  1. TotalRainfall.java: Compute the total rainfall by Zipcode and Year. Only one output record should be reported for each combination of Zipcode and Year. The output should have the following format:

    ZipcodeYearTotal-Rainfall
    …
    0000119491500.00
    0000119501400.00
    ...
  2. TopTenTemp.java: Identify the top ten temperature readings in the entire dataset. The output should contain ten records. The output should have the following format: Zipcode,Year,Month,Temp, as:

    …
    000011952AUG115.00
    000041963MAR110.00
    …

Deliverables (Send by email to the instructor in a single Zip file):

  1. The source code of the two MapReduce java programs. Use suggested names.
  2. The output of both programs executing the code with 2 reducers. Include the output of the first programs in a folder Output1 and the output of the second program in a folder Output2.

PART II (40%)

Using the Virtual Machine prepared for this activity, and the instructions prepared to configure your AWS account and execute a MapReduce job, execute the following MapReduce program, that we have been studying in class:

InstructionalSolutions.jar[AverageTempByStation]

Use two reducers and the Meteorological Station Data that we have been using. As part of the configuration process you will need an Amazon Credit code. You will receive this by email.

Deliverables (Send by email to the instructor):

Capture the VM screen showing the results of executing the Hadoop job and the output of the MapReduce job.
